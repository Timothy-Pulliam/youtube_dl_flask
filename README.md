# youtube_dl_flask
youtube_dl web app built on flask

## how to run

```
git clone git@github.com:Timothy-Pulliam/youtube_dl_flask.git
cd youtube_dl_flask
python3 -m venv venv
pip install -r requirements.txt
flask run
```

go to http://localhost:5000

## how to use
enter a url pointing to a youtube video. Videos must be smaller than 100MB.
